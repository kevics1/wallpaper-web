about=Lushan Efficiency Suite is a comprehensive QGIS plugin designed to streamline geospatial workflows for field geography studies. Core functionalities include:

    1. Define Map Frame Layer: Create cartographic extent area surface data as map frame layer (EPSG:32650)
    2. Hillshade Base Map & Contour Generation: Process DEM data to generate rendered maps, hillshade, and contour lines
    3. Data Download: Automatically download OSM data (transport nodes, peaks, roads, rivers, lakes, land use)
    4. Data Projection: Project layers to EPSG:32650 coordinate system
    5. Data Symbolization: Symbolize according to Lushan practicum mapping system
    6. Text Annotation: Add appropriate text labels for all layers

    The plugin features a user-friendly interface with error handling and modular design for easy maintenance and extension.
